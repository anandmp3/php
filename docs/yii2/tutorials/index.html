Tutorials

The Definitive Guide to Yii is the most comprehensive Yii documentation that gives the definitive description of every feature of Yii. It is being constantly updated to reflect the latest enhancements and changes. A PDF version of this tutorial can be found in the Yii releases.

Larry Ullman's Learning Yii Series is best suited for people who just start with MVC and/or Yii programming. (Difficulty level: 1)

The Yii Blog Tutorial describes in detail how to use Yii to develop a practical Web application. It explains the usage of Yii in a systematic way. You may also find a PDF version of this tutorial in the Yii releases. (Difficulty level: 3)

Books

The first Yii book is titled "Web Application Development with Yii and PHP". It takes a long-format tutorial approach to learning Yii. It is an introductory book that takes you step-by-step to develop a real world Web application with Yii, from initial concept through production-readiness. You may also find this to be a useful tutorial.

Visit the Book Website >

The second Yii book titled "Yii Application Development Cookbook" is a recipe book that contains 13 independent chapters full of useful recipes covering most interesting Yii parts. If you are a developer with a good knowledge of PHP5, are familiar with the basics of Yii, and have tried to develop applications using Yii, then this book is for you.

Visit the Book Website >

The third book on the Yii framework is titled, simply enough, "The Yii Book". Written by Larry Ullman, author of the popular "Learning the Yii Framework" blog series, this book explains the most important features and concepts in the Yii framework, using practical, real-world examples.

Visit the Book Website >

Besides the Yii books, we also recommend the following books for improving your PHP and Web development skills:

For beginners.

PHP 6 and MySQL 5 for Dynamic Web Sites

A well-written book that teaches you to quickly learn PHP and MySQL, even if you have no prior experience with them before.

For beginners.

jQuery in Action, Second Edition

An introductory book on jQuery and jQuery UI

For beginners.

SQL Cookbook

A complete reference book good for finding solutions for common SQL problems.

For the experienced.

Pro JavaScript Design Patterns

A book for experienced JavaScript developers who want to use JavaScript to cope with problems in more advanced and standard ways.

For the experienced.

Joe Celko's SQL for Smarties: Advanced SQL Programming Third Edition

A very nice book on solutions to some common yet complex SQL problems.

For the experienced.

Design Patterns: Elements of Reusable Object-Oriented Software

The classic design pattern book!

For the experienced.

Communicating Design: Developing Web Site Documentation for Design and Planning

A nice book describing how to plan and design a new Web application.
