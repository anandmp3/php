服务器要求

  • PHP 版本 5.1.6 或更新的版本。
  • 大多数 Web 应用程序需要数据库。当前支持的数据库为：MySQL (4.1+), MySQLi, MS SQL, Postgres, Oracle, SQLite 和 ODBC。
