创建适配器

适配器目录和文件结构

适配器目录和文件结构布局的例子：

  • /application/libraries/Driver_name
    • Driver_name.php
    • drivers
      • Driver_name_subclass_1.php
      • Driver_name_subclass_2.php
      • Driver_name_subclass_3.php

注意: 为了在大小写敏感的文件系统上维持兼容性，这个 Driver_name 目录必须使用 ucfirst() 函数处理。
